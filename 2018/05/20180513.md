# 数字化资产的定义

在当下的数字化资产里面，我个人跟他去做了这样一个划分，在数字化资产的代币的体系当中分为三个动作：**买币，发币，转币，** 基于以上的三点，我来跟大家来分析一下其中的运行的原理；



首先我们跟大家来谈一谈关于买币的问题，我们现在在玩数字化资产这件事情上面，本身来说有两种方式去买，一种方式叫做以法定货币去交换数字化资产，第二种的是以数字化资产去交换数字化资产，那么在完成这两件事情的时候，我们其实所需要去知道的是要到什么交易所上面去完成购买，或者直接在私下面去完成兑换都是可以的。

在这个维度上面，因为我们的所有的货币的发放都是在以太坊网站上面去发放的可记录的数字化的代币，那么在这个时候，我们其实每个个体都可以看到这数字化货币的流通的个体，以及这个货币交易的情况和这个货币所存在的价值，那么这是我们所能够非常清晰的看到的。



>我们有很多人在玩所谓的抄币，但是其实这一种思维体系本身就是错误的。

我们其实应该把所有的数字化资产都当成一种游戏化的资产，这种游戏化的资产，只是因为说在玩这种游戏的时候，这种游戏性的代币能够和真实化的代币相连接起来，在这个时候就完成商业的价值，其实关键所在不是在于这个代币本身的价值，而是这个代币背后有人在运作这件事情，这个代币的背后，能够去完成相应的产品的打造，完成相应的游戏的生产，完成相应的应用的打造，在这个时候才真正的具备价值。

所以我现在，对于它的一个定义来说，我把所有公司或者个人发行的代币，统一定义为游戏化的货币，这种游戏化的货币，相当于是某一种应用的游戏化的点数，**而这一种游戏化的点数的价值所在是在于能不能去打造一款有价值性的游戏出来，只要我们能创造出有价值性的游戏，那么这个时候我们就为我们的游戏的货币赋予的价值。**

# 代币的本质

那么我们在进行所谓的代币的过程当中，那么所需要去知道的关键所在是需要去完成代币购买，在玩代币的购买的时候，那么在当下购买的早期的时候有空投和赠送，那么随着时间的不断推移，更多的情况下就要通过交易所来去完成购买，而现在我们所看到的完成交易所购买的过程当中，真正非常的舒服的一种购买方式，**使用交换的方式，而不是用一些数字化的k线图的方式去完成购买，这样的形式会做得更好一点，所以其实我们在完成数字化资产的购买的时候，我们其实是和所有所有持有这个数字化资产的个体去达成一种交易，而和所有的数字化资产达成一种交易的过程中，是有一部分的机构去完成的数字化资产的交换，有一部分的个人去完成的数字化资产的交换，那么在这个交换的过程中，我们其实就是用我们所持有的法币和个人和机构去完成交易，而不是和一个机构去完成交易，** 这是我们所应该去明白的一个思维体系，而现在很多的交易所都在逐渐的去淡化这种过程，他想要是直接让用户和交易所去完成交易，而不是用户和用户去完成交易，在这一点上面我个人觉得otcbtc，做的是最好的。



所以在这个货币交换的过程当中，大家其实就会非常清晰的看到，我们其实本身来说不是在和某种机构在打交道，而是在和某个代表机构的个人的人在打交道，因为这一个代币的本质上面来说，是这个人或者这个机构完成交易，而我们其实是人和人之间在完成一种交流，**在完成人和人之间去完成一种交流的时候，那么这个时候形成的一种共识，从而去完成了货币的交易。**
